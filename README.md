## üìÇ Project Structure
When you first run the program, it will ask you to log in.
You can find the default login credentials in the file Users.txt.

After logging in seccessfully, the Main Screen will appear, containing 10 options:

1. Show Clients List - Display all clients with their details.
2. Add New Client - Allows you to add a new Client ( as long as they are not already registered ).
3. Delete Client - Enables you to delete an existing client.
4. Update Client Info - Lets you update a client's information.
5. Find Client - Allows you to search for a specific client.
6. Transactions - Opens a submenu with options:
     1. Deposit
     2. Withdraw
     3. Total balances
     4. Transfer
     5. Transer log
     6. or return to main menue.
7. Manage Users - Opens another menu where you can view following:
     1. List Users
     2. Add New User
     3. Delete User
     4. Update User
     5. Find User
     6. Main Menue
8. Login Register - Displays login records showing when each user logged in, their username, and their permissions.
9. Currency Exchange - Opens a submenu with the following options:
     1. List Currencies
     2. Find Currency
     3. Update Rate
     4. Currency Calculator
     5. Main Menue
10. Logs you out and returns you to the login screen.
